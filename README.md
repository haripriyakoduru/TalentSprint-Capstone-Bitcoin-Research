# Bitcoin Macro Volatility Research (2024-2026)
This repository contains the source code, dataset, and final research paper for the Accenture TalentSprint Capstone Project.

## Project Overview
An analysis of Bitcoin's pro-cyclical response to FOMC and CPI announcements following the 2024 ETF institutionalization.

## Contents
- `Research Paper.ipynb`: Data cleaning and OLS Regression.
- `Master_Research_Data.csv`: The final synchronized master dataset generated by the script.
- `Capstone_Research_Paper__Final_.pdf`: The 16-page IEEE research document.

### Input Data (CSV)
To run the analysis, the following individual CSV files are required in the root directory:
- `Bitcoin Research - Bitcoin (BTC-USD).csv`: Historical daily price data for Bitcoin.
- `Bitcoin Research - S&P 500 (^GSPC).csv`: Historical daily price data for the S&P 500.
- `Bitcoin Research - ETF Flow.csv`: Daily net inflows/outflows for Spot Bitcoin ETFs.
- `Bitcoin Research - FOMC.csv`: Historical dates for Federal Open Market Committee meetings.
- `Bitcoin Research - CPI.csv`: Historical dates for Consumer Price Index releases.
